# 🏫 校园端管理后台

校园AI助手 — 教师管理后台

## 快速启动

```bash
cd school-campus
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

或双击 `start.bat`（Windows） / `start.ps1`。

打开浏览器访问: **http://localhost:8000**

## 功能

- **🔐 登录/注册** — 教师账号注册和登录
- **🤖 AI 助手** — 通过自然语言对话完成所有管理操作
- **📋 班级管理** — 创建/删除班级
- **📚 课程管理** — 创建/删除课程
- **📍 签到管理** — 发起签到（生成6位签到码）/ 查看签到记录
- **📝 作业管理** — 发布作业 / 查看学生提交 / 批改评分
- **📋 请假审批** — 审批学生请假申请
- **📢 公告管理** — 发布全校/班级公告
- **🔄 实时活动** — 右侧面板实时显示学生活动

## AI 指令示例

| 你说 | 系统执行 |
|:---|:---------|
| 创建班级「高一(1)班」 | 在数据库创建班级 |
| 创建课程「高等数学」 | 创建课程记录 |
| 发起签到 | 生成6位签到码 |
| 发布作业「课后习题」 | 发布新作业 |
| 审批请假 | 列出待审批请假 |
| 查看班级列表 | 列出所有班级 |
| 查看课程列表 | 列出所有课程 |

## 技术栈

- **后端**: FastAPI + SQLite + SQLAlchemy
- **前端**: 纯静态 HTML + CSS + JS（零构建）
- **端口**: 8000
